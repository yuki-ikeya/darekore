# だれコレ

__ハッカソン__（Open Hack U 2025 TOKYO）にて開発したチームプロジェクトです。  
https://hacku.yahoo.co.jp/hacku2025_tokyo/<br>
<br>

__デモ__<br>
https://darekore-ocha.web.app<br>
<br>

__概要__<br>
ユーザが写真に写った人物に関する情報を整理・検索することを目的としたWebアプリケーションです。  
すべての処理が端末内で完結するため、プライバシーに配慮しながら人物情報を保存・管理できます。
<br>
<br>

__担当・役割__<br>
本プロジェクトはハッカソンにてチームで開発しました。  
私は主に以下を担当しました：

- フロントエンド実装
- UI/UX設計
- プレゼンテーション
- 進行管理（役割分担・タイムキープ）

メンバーの役割分担や開発スケジュールの管理を行い、
限られた時間の中で効率的に開発を進めました。

また、ユーザ視点を意識した画面設計を行い、
直感的に操作できるインタフェースの実装に取り組みました。<br>
