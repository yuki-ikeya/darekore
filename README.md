## Demo
https://darekore-ocha.web.app


## 概要
ユーザが「誰か」を特定・共有することを目的としたWebアプリケーションです。  
（※ここは内容に合わせて少し変えてもOK）


## My Role（担当・役割）

本プロジェクトはハッカソンにてチームで開発しました。  
私は主に以下を担当しました：

- フロントエンド実装
- UI/UX設計
- プレゼンテーション
- 進行管理（役割分担・タイムキープ）

チームリーダーとして、メンバーの役割分担や開発スケジュールの管理を行い、
限られた時間の中で効率的に開発を進めました。

また、ユーザ視点を意識した画面設計を行い、
直感的に操作できるインタフェースの実装に取り組みました。
